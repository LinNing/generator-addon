# generator-addon


> Yeoman 生成器，用于生成基于 mAuthor 的前端组件addon结构。


## 安装

- **安装 yo**

    ```bash
    $ npm install -g yo
    ```

- **安装 generator**

    ```bash
    $ npm install -g generator-addon
    ```

## 使用

### 组件目录结构

- **生成**

    ```bash
    $ yo nd
    ```

- **说明**

    ```
    |-<name> <外部目录，非本脚本生成>
        |-css.scss
        |-dependencies.xml
        |-model.json
        |-presenter.js
        |-preview.html
        |-view.html
    ```
